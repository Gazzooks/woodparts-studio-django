Pilot Hole Tips:
  • For softwoods, use a pilot hole with a diameter of 50-70% of the screw’s root diameter.
  • For hardwoods, use a pilot hole with a diameter of 70-90% of the screw’s root diameter.
  • For wood screws, the pilot hole should be as deep as the threaded portion will penetrate.
  • For a clearance hole, drill a hole slightly larger than the screw’s outside diameter.
  • When screwing near the edge, always use a pilot hole to prevent splitting.

Thread Information:
Machine screws typically come in coarse (UNC) and fine (UNF) thread versions. The number after the dash in a machine screw size indicates threads per inch. For example, #8-32 is a #8 screw with 32 threads per inch (coarse thread).
